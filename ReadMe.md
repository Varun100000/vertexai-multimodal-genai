# 🎯 Multimodal GenAI with Vertex AI (Text, Audio, Video)

## 🚀 Overview

This project demonstrates how to build **multimodal Generative AI applications** using **Vertex AI and Gemini models**, capable of processing and generating:

- 📝 Text
- 🎧 Audio
- 🎥 Video
- 🔀 Combinations of multiple modalities

The notebook showcases how different data types can be unified into a single AI workflow.
